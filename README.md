# R Learning Website

This repository contains the source files for an illustrative R Learning Website created to accompany the blog post "Building Your First Website with Quarto: A Step-by-Step Guide."

The project serves as an example for demonstrating the complete workflow of creating, structuring, rendering, and publishing a website using Quarto. It is intended for learning purposes and is not meant to be a fully developed educational website.
